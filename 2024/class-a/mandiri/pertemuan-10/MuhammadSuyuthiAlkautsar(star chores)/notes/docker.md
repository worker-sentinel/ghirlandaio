
# apa itu docker? 
alat untuk memudahkan pengembangan dan deployment aplikasi dengan membungkus kode serta dependasinya ke dalam kontainer

memudahkan komunikasi antara developer dan operator, tidak perlu install aplikasi, server dll karena sudah dikemas dalam container, karena human error bisa saja terjadi

# docker vs VM
