
# apa itu docker? 
alat untuk memudahkan pengembangan dan deployment aplikasi dengan membungkus kode serta dependasinya ke dalam kontainer

memudahkan komunikasi antara developer dan operator, tidak perlu install aplikasi, server dll karena sudah dikemas dalam container, karena human error bisa saja terjadi

# docker vs VM

docker mempunyai application layer


VM meempunyai application layer dan OS kernel

# Konsep utama docker


### Docker images
Paket yang berisi kode aplikasi, runtime, library, dan konfigurasi lingkungan. Ini adalah cetak biru untuk menjalankan aplikasi

### container
Instansi yang berjalan dari sebuah image

### Docker registry
untuk mengundug docker image
