# Apa Perbedaan Antara Open Source dan Closed Source
## Latar Belakang
Perkembangan teknologi pada era digital saat ini membawa perubahan besar bagi berbagai aspek kehidupan manusia. Terutama dalam penggunaan perangkat lunak (software). Perangkat lunak menjadi komponen paling penting yang mendukung berbagai aktivitas manusia. Seiring dengan berkembangnya teknologi tersebut muncu berbagai model pengembangan perangkat lunak yang digunakan oleh para pengembang.
Saat ini telah terjadi peningkatan minat terhadap open source sebagai paradigma baru dalam pengembangan software. Open source merujuk pada penggunaan kode sumber yang dibagikan, standar terbuka, serta kolaborasi antar para pengembang software dan pengguna di seluruh dunia untuk membangun perangkat lunak, mengidentifikasi, memperbaiki kesalahan, dan melakukan peningkatan. Paradigma tradisional yang disebut closed source memandang pengembangan perangkat lunak sebagai proses yang sangat spesial yang paling baik dikelola oleh tim pengembang yang berkualifikasi tinggi dengan manajemen proyek yang cermat, serta peningkatan yang dilakukan sesekali dalam bentuk rilis versi baru. (Mishra et al., 2002)
Secara umum perangkat lunak dapat dibedakan menjadi dua jenis yaitu open source dan closed source. Kedua perangkat ini memiliki kelebihan dan kekurangan masing-masing serta digunakan secara luas di berbagai bidang. Oleh karena itu sangat penting untuk memahami perbedaan antara closed source dan open source.

## Isi 
Open source/free open source software adalah istilah yang memiliki tiga makna yaitu cara pengembangan software, jenis lisensi software, dan model bisnis software. Istilah open source merupakan sebuah program yang pembuatnya memberi izin kepada pengguna menjalankan program itu untuk apa saja, mempelajari dan memodifikasi program itu dan menyebarluaskan copy program asli atau program yang sudah dimodifikasi tanpa harus membayar royalti kepada pengembang pengembang sebelumnya. 
Sedangkan closed source yaitu program yang kode sumbernya dirahasiakan oleh pengembang. Proprietary dan hak menyebarluaskan program proprietary hanya pada pemilik hak cipta. Orang lain harus mendapatkan izin dari pemilik hak cipta untuk menggunakan, memodifikasi, dan menyebarluaskan program proprietary.
Open source dan closed source bisa dikatakan kebalikan karena pemilik hak cipta open source tidak melarang penggunaan, pemodifikasian, dan penyebarluasan program ciptaannya atau pemilik hak cipta tidak menyediakan kode sumber sehingga tidak boleh disebarluaskan makan program itu bukan open source. (Maryanto, 2014)
Berdasarkan pengertian diatasm, terdapat beberapa perbedaan antara open source dan closed source yang dapat dilihat dari berbagai aspek. Linux sebagai representasi open source dan Windows sebagai representasi closed source menjadi contoh umum yang digunakan dalam membandingkan kedua konsep tersebut. (Hussain et al., 2015)
1.	Akses penuh dan tanpa akses

2.	Kebebasan lisensi dan pembatasan lisensi
Linux memungkinkan pengguna untuk memodifikasi, menggunakan, mendistribusikan, dan menjual software selama kode masih terbuka. Sedangkan windows membatasi penggunaan berdasarkan jumlah lisensi yang dibeli.
3.	Dukungan komunitas dan dukungan berbayar
Linux didukung oleh komunitas besar melalui forum dan internet. Pengguna juga bisa membeli ukungan dari Perusahaan seperti red hat. Windows juga memiliki banyak forum tetapi, pengguna dapat membeli dukungan resmi dari Microsoft yang dianggap lebih aman oleh Perusahaan.
4.	Dukungan hardware penuh dan sebagian
Dulu linux sulit digunakan karena keterbatasan hardware. Sekarang sudah jauh lebih baik meskipun masih terdapat bebrapa kendala. Windows umumnya mendukung hampir semua hardware meskipun terkadang pengguna harus mencari-cari secara manual.

6.	Instalasi aplikasi terpusat dan tidak terpusat
Linux memiliki sistem manajemen paket terpusat yang memudahkan installasi aplikasi. Sedangkan windows mengaruskan pengguna mengunduh dan menginstall aplikasi secara manual.
7.	Fleksibilitas dan keterbatasan
Linux sangat fleksibel dan bisa dikustomisasi sesuai keinginan pengguna, sedangkan windows lebih terbatas karena menggikuti aturan dari Microsoft.
