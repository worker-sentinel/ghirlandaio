
# Server dan data center
bentuk fisik server yang biasanya diletakkan dalam rak di ruang data center. Server berfungsi sebagai pusat pemrosesan data

# Node
Node dapat berupa server fisik tunggal atau unit virtual di dalam sebuah server fisik


# Cluster
Cluster adalah sekumpulan server yang bekerja bersama secara paralel untuk menangani beban kerja yang besar

# Cara Kerja Cluster
Dalam sebuah cluster, terdapat controller (seperti load balancer) yang bertugas membagi-bagi request atau beban kerja ke berbagai server (misalnya web server atau database server)

# Pentingnya Klastering
Penggunaan cluster sangat penting agar layanan publik atau situs web tetap stabil saat menerima banyak request sekaligus
