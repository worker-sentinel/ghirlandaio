## 1. Install Ubuntu 
Download ISO 
```
https://ubuntu.com/download/server/thank-you?version=26.04&architecture=amd64&lts=true
```
## 2. Buka Aplikasi VirtualBox / VMware
Jalankan aplikasi virtualisasi yang terpasang, seperti Virt-Manager, Lalu buat VM Baru Klik tombol New, kasih nama bebas, dan pilih tipe OS Linux (Ubuntu 64-bit).

## 3. Atur Spek Komputer Virtual
Atur ukuran RAM (minimal 2 GB), jumlah CPU (minimal 2 Core), dan kapasitas harddisk (minimal 20 GB).

## 4. Masukkan Kaset ISO (CD Virtual)
Masuk ke menu Settings VM, pilih bagian Storage, lalu masukkan file ISO Ubuntu yang tadi udah didownload

## 5. Nyalakan dan Mulai Install
- Klik tombol start (Mulai) untuk menyalakan komputer virtual.
- Pilih opsi try or install Ubuntu server, lalu ikuti setup di layar (pilih bahasa, isi username, dan password) sampai selesai
