# INSTALL ATOM

```
ls
```
> Untuk cek file/folder apa saja yang ada

```
cd atom/
```
> Masuk ke direktori ATOM

```
ls
```
> Untuk cek file/folder apa saja yang ada

```
docker ps
```
> Untuk cek kondisi container

```
docker compose up -d
```
> Menjalankan semua service untuk menghidupkan ATOM

```
docker compose exec atom npm install
```
> Untuk menginstall ATOM

```
docker compose atom npm run build
```
>  Untuk menyusun ulang semua file install agar tampilan ATOM bisa keluar dengan benar di browser

```
exit
```
