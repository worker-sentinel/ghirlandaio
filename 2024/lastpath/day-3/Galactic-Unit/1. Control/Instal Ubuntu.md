# Instalasi Ubuntu Server

**Unduh Ubuntu Server ISO** 
Download file ISO Ubuntu Server dari situs resmi Ubuntu dan simpan pada direktori yang mudah diakses.
Link untuk mendownload sebagai berikut
   ```
   https://ubuntu.com/download/server/thank-you?version=26.04&architecture=amd64&lts=true
   ```

**Buka Virtual Machine**  
Jalankan aplikasi virtualisasi seperti VirtualBox atau VMware.

**Buat Virtual Machine Baru**  
Pilih menu **Create New** untuk membuat mesin virtual baru, kemudian masukkan nama dan konfigurasi dasar sesuai kebutuhan.

**Pilih File ISO**  
Pada bagian media instalasi, pilih file ISO Ubuntu Server yang telah diunduh sebelumnya untuk digunakan sebagai boot installer.

**Lanjutkan Proses Instalasi**  
Atur alokasi RAM, CPU, dan penyimpanan, kemudian jalankan virtual machine dan ikuti proses instalasi hingga selesai.
