### untuk memulai merekam
```
asciinema rec [nama file baru].cast
```

### untuk mengentikan rekaman 
```
ctrl + D
```

### untuk memutar hasil rekaman 
```
asciinema
```

### untuk mengupload di website (agar bisa di download hasil rekamannya)
```
asciinema upload [nama file recordnya].cast
```
setelah itu buat akun di asciinema.org
