***Error PHP***

Buat masuk ke website Arteri, pertama cek dulu IP lewat ```ip a``` di terminal/TTY. Device yang dipakai harus terhubung ke Wi-Fi atau jaringan yang sama. Setelah itu copy IP-nya, lalu paste ke browser di device lain buat buka website Arteri. Tapi pas dicoba, masih muncul beberapa error PHP dan error lainnya 

Intinya sih error itu muncul karena website Arteri masih pakai framework jadul (CodeIgniter lama), sedangkan laptop saya udah pakai PHP versi baru yang lebih strict. Jadi ada beberapa kode lama yang sekarang udah nggak cocok, makanya muncul error.
