## Audience

Anda diminta oleh atasan anda mempersiapkan server untuk aplikasi perpustakaan.  Anda di haruskan mengimplementasikan protokol CIS untuk server dari referensi distro lain.Dalam proses installasi, anda tidak diberikan akses SSH, jadi anda harus melakukan deployment secara langsung. Anda akan melakukan deployment dalam mode native.

### Kelas A

- Aplikasi yang di install adalah openkm
- menggunakan nginx untuk reverse proxy
- Impelementasi CIS Server secara keseluruhan
- Impelementasi CIS Docker
- Aplikasi bisa diakses secara dalam jaringan local
- Firewalld telah terkonfigurasi dengan tepat
- Disable module kernel yang tidak diperlukan
- Hardening pada layer kernel
- metode yang digunakan adalah metode CLI 
- Dilarang menggunakan package base-devel, jika di butuhkan aplikasi pendukung install secara mandiri di luar package meta base-devel
- meggunakan kernel linux-lts
- database dan aplikasi berada di laptop yang berbeda
- menggunakan dokcker dan docker swarm
- setiap orang memahami setiap alur pekerjaan, 
- diharuskan menggunakan project pada github project,
- pemahaman terhadap langkah yang diguanakan, tidak hanya sebatas eksekusi
