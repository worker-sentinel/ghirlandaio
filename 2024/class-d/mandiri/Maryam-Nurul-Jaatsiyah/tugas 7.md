## Mata Kuliah: Perpustakaan dan Arsip Digital

1. Boot (Booster) : bootloader sebagai awalan proses booting. Mengubah bahasa manusia (huruf atau angka) untuk bisa dibaca prosesor. 
2. Luks on Lvm: partisi dienkripsi dengan LUKS di atas LVM, supaya data lebih aman. Enkripsi yang melindungi data pada hard disk sehingga data tidak dapat diakses tanpa kata sandi. 
3. Desktop (XFCE): tampilan antarmuka grafis untuk memakai Linux seperti desktop biasa. 
4. Superfile: pengelola berkas berbasis terminal (CLI)  untuk membuka, memindah, menyalin, dan menghapus file. 
5. CIS: standar keamanan yang berisi pedoman konfigurasi sistem operasi. Dalam instalasi Arch Linux, standar CIS digunakan sebagai acuan untuk menentukan pengaturan partisi, hak akses, kebijakan kata sandi agar sistem lebih aman. 
6. Container (Podman):menjalankan dan mengelola container tanpa Docker. Podman Desktop menyediakan antarmuka grafis yang memudahkan pengguna dalam membuat, menjalankan, dan mengelola container.
7. Firewall: mengatur pintu mana yang dibuka dan yang mana yang ditutup. 
8. Multimedia (MPC, MPD, MPV): MPD untuk mengelola musik, butuh jembatan untuk menjalankannya. MPV untuk media, dan MPC menjadi jembatan untuk menjalankan perintah. Bisa nonton YouTube tanpa iklan.
9. Keepass xc: aplikasi pengelola kata sandi.  Fiturnya lengkap. Aplikasi ini membantu pengguna mengelola banyak akun, membuat kata sandi yang kuat.
10. Secret: lebih simpel. Untuk menyimpan atau mengelola kunci rahasia.
11. Openssh: akses jarak jauh yang aman ke server atau komputer melalui SSH. Mengakses komputer atau server dari jarak jauh secara aman.
