## Kernel
inti dari sistem operasi yang berfungsi menghubungkan perangkat keras (hardware) dan perangkat lunak (software), serta mengatur penggunaan sumber daya komputer seperti CPU, memori, dan penyimpanan.

## LVM (Logical Volume Management) 
digunakan untuk mengelola penyimpanan agar lebih fleksibel. Dengan LVM, kapasitas penyimpanan dapat dibagi menjadi beberapa logical volume sesuai kebutuhan.

## LUKS (Linux Unified Key Setup) 
digunakan untuk mengenkripsi penyimpanan sehingga data lebih aman dan tidak dapat diakses oleh pihak yang tidak memiliki kunci enkripsi.

## KeePassXC dan Secrets 
digunakan untuk menyimpan password, SSH key, authenticator, dan credential lainnya secara aman sehingga tidak perlu disimpan dalam file teks biasa.

## OpenSSH 
digunakan untuk melakukan remote access ke server atau komputer lain melalui jaringan dengan koneksi yang terenkripsi dan lebih aman.

## Remote Access 
memungkinkan komputer workstation terhubung ke komputer server tanpa harus berada di depan perangkat secara langsung.

## CLI (Command Line Interface) 
banyak digunakan dalam administrasi server karena lebih ringan, cepat, dan memberikan kontrol yang lebih luas dibandingkan GUI.

## Network Manager 
membantu mengelola koneksi jaringan seperti Wi-Fi dan Ethernet agar konfigurasi jaringan lebih mudah dilakukan.

## Firewall 
digunakan untuk mengatur lalu lintas jaringan dan membatasi akses yang tidak diperlukan ke dalam sistem.

## Iptables 
tools firewall pada Linux yang digunakan untuk mengatur aturan jaringan, membuka atau menutup port tertentu, serta meningkatkan keamanan server.

## Podman 
digunakan untuk menjalankan aplikasi di dalam container sehingga aplikasi dapat berjalan secara terisolasi dan mengurangi konflik antar aplikasi.

## XFCE4 
desktop environment Linux yang ringan dan tidak membutuhkan banyak sumber daya komputer sehingga cocok untuk perangkat dengan spesifikasi terbatas.

## Superfile 
file manager berbasis terminal yang memudahkan pengelolaan file dan direktori tanpa menggunakan antarmuka grafis.

## Booster 
digunakan untuk membantu proses booting sistem dengan memuat driver dan konfigurasi yang diperlukan saat sistem dijalankan.

## MPD (Music Player Daemon) 
berfungsi sebagai layanan pemutar musik yang berjalan di latar belakang.

## MPC (Music Player Client) 
digunakan untuk mengontrol MPD melalui terminal.

## MPV 
media player yang digunakan untuk memutar file audio maupun video.

## CIS (Center for Internet Security) 
berisi panduan dan rekomendasi konfigurasi keamanan agar sistem lebih aman sesuai praktik keamanan yang baik.

Pada penjelasan pertemuan dikelompok 7 juga menekankan pentingnya keamanan data, penggunaan password manager, enkripsi penyimpanan, serta otomasi sistem untuk mempermudah pengelolaan server dan mengurangi risiko keamanan.
