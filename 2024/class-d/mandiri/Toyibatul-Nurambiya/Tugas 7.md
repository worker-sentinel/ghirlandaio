## LUKS on LVM
LUKS on LVM adalah metode enkripsi disk dalam sistem Linux yang. Menggabungkan teknologi LUKS (Linux Unified Key Setup) dengan LVM (Logical Volume Manager). LVM digunakan untuk menciptakan dan mengelola volume logis terlebih dahulu. Kemudian volume tersebut dienkripsi dengan LUKS agar data di dalamnya terlindungi. Terlindungi dari akses ilegal.

## Superfile
Superfile adalah pengelola file berbasis antarmuka perintah (CLI) yang digunakan di sistem operasi berbasis terminal. Terminal, yang dirancang untuk sistem Linux. Superfile memungkinkan pengguna mengelola file dan direktori langsung melalui terminal.

## CIS
CIS (Center for Internet Security) untuk mengamankan folder penting seperti home, tmp, atau var. Folder tersebut dipisahkan supaya file sementara tidak hilang walau laptop dalam keadaan terestart.

## Podman, Podman Desktop
Podman adalah engine container yang digunakan untuk membuat dan menjalankan aplikasi dalam wadah. Container di sistem Linux. Sedangkan Podman Desktop adalah aplikasi antarmuka grafis yang digunakan untuk mempermudah pengelolaan container Podman. Melalui aplikasi ini, pengguna dapat membuat, menjalankan, menghentikan, dan memantau container. Menggunakan tampilan visual tanpa perlu selalu memakai perintah di terminal.

## Ip Tables
Ip tables adalah alat bawaan di Linux yang berfungsi sebagai firewall atau benteng yang melindungi sistem dari akses yang tidak sah. Fungsi keamanan jaringan pada sistem operasi linux adalah untuk memeriksa dan memastikan keamanan serta integritas data yang berada dalam sistem tersebut. Mengendalikan dan mengatur setiap data atau akses internet yang masuk dan keluar. Dengan menggunakan ip tables ini dapat mengamankan jaringan dengan cara yang lebih mudah dan efektif. Memperbolehkan data yang aman dan dapat dipercaya, atau bisa memblokir secara langsung akses yang mencurigakan untuk mencegah serangan siber.

## MPD, MPC, MPV
MPD (Music Player Daemon) adalah mesin pemutar musik yang berjalan di belakang layar. Berada di belakang dan bertugas memutar file audio, mengelola playlist, serta menyimpan basis data. Data musik. Untuk mengoperasikan MPD diperlukan MPC (Music Player Client), yaitu Aplikasi klien yang digunakan untuk menjalankan perintah seperti memutar, menangkap, dan mengatur alur kerja sistem. Menghentikkan, atau mengganti lagu. Sementara, MPV adalah pemutar media yang dapat digunakan untuk memutar berbagai format audio dan video, sehingga selain itu juga bisa menampilkan berbagai jenis file multimedia lainnya. Untuk mendengarkan musik, fitur tersebut juga bisa digunakan untuk memutar video.

## Keepass XC
Keepass adalah format file basis daya yang berfungsi sebagai brangkas digital lokal untuk menyimpan semua password dengan aman tanpa ketergantungan internet.

## Secrets
Secrets adalah sistem yang digunakan untuk menyimpan informasi sensitif seperti kata sandi. Dengan adanya sistem ini, aplikasi dapat digunakan oleh pengguna tanpa harus memasukkan kata sandi berulang kali.

## OpenSSH
OpenSSH (OpenBSD Secure Shell) adalah program yang digunakan untuk membuat koneksi aman antara komputer. Koneksi atau komunikasi yang aman melalui jaringan menggunakan protokol SSH. (Secure Shell). Dengan OpenSSH, pengguna bisa mengakses perangkat dari jarak jauh, yaitu dapat mengakses dan mengontrol komputer lain.

## Booster
Booster adalah alat pembuat initramfs modern yang menggunakan bahasa Go, dan berfungsi untuk memuat driver enkripsi dasar tepat saat komputer dihidupkan.

## CLI
CLI (Command Line Interface) adalah cara untuk berinteraksi dengan komputer menggunakan perintah tulisan yang diketik di layar terminal atau command prompt, bukan dengan menggunakan tampilan grafis seperti yang biasa ditemui pada antarmuka pengguna.
