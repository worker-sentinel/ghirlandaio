## Booster 
Bisa membuat suatu file kecil untuk memulai sistem operasinya menjadi mulai atau bekerja

## Keepassxc
Password untuk server. aketika mengolah suatu server, tiap server harus memiliki password yang berbeda. Berfungsi sebagai manajemen passworsd

## Secreet 
Berfungsi otomatisasiin SSH key-nya ketika nge-push langsung, jadi tidak usah manual

## Superfile
Windows explorer, ketika menggunakan superfile, kita bisa langsung mengakses file-explorer dari terminal tanpa ada tampilan apk windows explore yang bisa dijalankan dari terminal dan metode mngoperasikan menggunakan command.
## Metode pengoperasian komputer ada 2 jenis
1. GUI (Graphical User Interface) di android pakai GUI. Contohnya ketika ingin menghapus foto kemudia ada pop up bertulsikan pertanyaan "Apakah ingin menghapus foto tersebut?" kemudia klik Yes
2. CLI (Command Line Interface) seluruh pengoperasian di terminal menggunakan command.

## iptables 
Firewall di jaringan. Setiap sistem operasi, firewall sebagai penjaga pintu atau biasanya dikenal di komputer itu port. Setiap aplikasi menggunakan port sebagai akses
 ## Podman 
 Memeprmudah pengerjaaan untuk install aplikasi. Jadi tidak ada masalah port conflict karena podman sudah didesain hal tersebut terjadi. Podman di dalamnya sudah di konfig oleh developer-nya. Podman bisa selesai dalam satu command untuk men-deploy atau install aplikasi. Contoh Install SLIMS itu caranya podman, run, SLIMS

 ## MPD, MPC, dan MPV
 mpc adalah klien baris perintah untuk Music Player Daemon (MPD). Aplikasi ini terhubung ke MPD dan mengontrolnya sesuai dengan perintah dan argumen yang diberikan.
 Mpv adalah pemutar media yang berbasis pada Mplayer  mendukung berbagai macam format file video, codec audio dan video, serta jenis subtitle.

 ## OpenSSH
 OpenSSH (OpenBSD Secure Shell) adalah seperangkat program komputer yang menyediakan sesi komunikasi terenkripsi melalui jaringan komputer menggunakan protokol Secure Shell (SSH).

 ## LVM on LUKS 
LVM pada konfigurasi LUKS (mengenkripsi partisi disk mentah dan membuat volume logis di dalam kontainer yang telah didekripsi) adalah metode standar untuk enkripsi disk penuh di Arch Linux. Ini memberikan keamanan maksimum sambil mempertahankan kemampuan pengubahan ukuran yang fleksibel dari LVM.

 ## CIS
 Administrator dapat memastikan bahwa sistem yang dikelola memiliki tingkat keamanan yang lebih baik dan konfigurasi yang konsisten di seluruh server.

 ## Cryptsetup
 Aplikasi untuk mengenkripsi data. Enkripsi adalah suatu metode untuk menyembunyikan informasi. Cryptsetup befungsi untuk mengenkripsi harddisk.
