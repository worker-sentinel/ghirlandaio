# Rangkuman
## LVM on LUKS
Partisi dienkripsi dulu dengan LUKS, lalu dibuat LVM di dalamnya. Praktis karena cukup membuka satu enkripsi untuk mengakses semua volume
## LUKS on LVM
LVM dibuat dulu, kemudian setiap logical volume dienkripsi sendiri-sendiri. Lebih aman karena tiap volume memiliki enkripsi terpisah
## KeePassXC dan Secret Service
Tools untuk menyimpan dan mengelola password secara aman. Password dapat digunakan sebagai kunci enkripsi maupun autentikasi sehingga pengguna tidak perlu mengingat banyak password
## Podman
Platform container yang digunakan untuk menjalankan aplikasi secara terisolasi. Membantu menghindari konflik antar aplikasi dan mendukung otomatisasi layanan
## XFCE4
Desktop environment Linux yang ringan dan hemat resource, cocok untuk komputer dengan spesifikasi rendah atau server yang membutuhkan tampilan grafis
## Superfile
File manager berbasis terminal yang memudahkan pengelolaan file seperti menyalin, memindahkan, mengganti nama, dan menghapus file
## Iptables
Firewall Linux untuk mengatur lalu lintas jaringan. Digunakan untuk membuka atau menutup akses port demi meningkatkan keamanan server
## CIS
Standar keamanan yang berisi rekomendasi konfigurasi agar sistem lebih aman dan sesuai praktik keamanan terbaik
## OpenSSH
Tools untuk mengakses server dari jarak jauh secara aman menggunakan koneksi terenkripsi. Juga dapat digunakan untuk transfer file
## Booster
Pembuat initramfs yang membantu kernel menyiapkan driver dan konfigurasi saat proses booting, termasuk dukungan partisi terenkripsi LUKS
## MPD, MPV, dan MPC
- MPD: Layanan pemutar musik yang berjalan di background
- MPC: Client untuk mengontrol MPD melalui terminal
- MPV: Media player untuk memutar musik dan video.
